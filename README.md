# Bus Ticket Booking System 🚌

A simple Bus Ticket Booking System built using Python and Streamlit.

## Features
- Add new buses
- View available buses
- Book seats
- Cancel booked seats
- View passenger details
- Payment mode selection (Cash / Card / UPI)
- Seat availability tracking
- Data stored using JSON files

## Technologies Used
- Python
- Streamlit
- JSON
- Object Oriented Programming (OOP)

## Files in Project
- `app.py` → Main application code
- `buses.json` → Stores bus information
- `passengers.json` → Stores passenger details

## How to Run the Project

1. Install Streamlit:

```bash
pip install streamlit
```

2. Run the application:

```bash
streamlit run app.py
```

## Project Output
Users can:
- Add buses
- Book seats
- Cancel bookings
- View passengers
- Manage seat reservations
